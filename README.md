# 📊 SENF Education Impact Dashboard

A live real-time dashboard tracking education program reach 
across Afghanistan — from TV broadcast to AI chatbots.

## 🚀 Live Demo
🔗 (https://senf.af/education-dashboard)

## 🛠️ Tech Stack
- HTML, CSS, Vanilla JavaScript
- Chart.js — data visualizations
- Google Sheets — live database
- Google Apps Script — API layer (JSONP)
- Syne + DM Sans + Bebas Neue — typography
- LocalStorage — session persistence

## 📈 What It Tracks
- 📡 TV & Radio broadcast reach
- 💬 Lalah AI Chatbot — 153K+ users
- 📱 SENF App, Kankor App, Dana App
- 📊 Monthly growth trends per platform

## ⚡ Key Features
- One-click sync from Google Sheets
- Expandable charts per platform
- 8 months of historical data
- Password protected access
- Zero backend, zero framework
